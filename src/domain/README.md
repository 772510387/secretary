# Domain

`domain` 是系统可信核心。

这里的代码应该是纯函数或纯对象模型，输入相同则输出相同，方便测试和回放。

## 领域模块

- `market`：行情与交易日历模型。
- `portfolio`：账户、现金、持仓和流水。
- `trading`：交易意图和订单生命周期。
- `risk`：硬风控。
- `memory`：记忆读写策略。
- `cerebellum`：小脑事件和触发规则。
- `brain`：模型抽象和结构化协议。
- `research`：深度研究输出协议。
- `notification`：告警等级、冷却、去重。
- `audit`：审计事件模型。

## 实现原则

- 不关心数据从哪里来。
- 不关心数据写到哪里去。
- 不关心哪个模型供应商。
- 不关心哪个券商接口。
- 只关心业务规则是否正确。
