# Change Checklist

## 通用

- [ ] 读过 `AGENTS.md`。
- [ ] 读过目标模块 `README.md`。
- [ ] 改动范围和目标一致。
- [ ] 没有引入无关重构。
- [ ] 更新了相关 README 或说明无需更新。

## 资金和交易

- [ ] 金额、数量、成本价、盈亏有测试。
- [ ] T+1、100 股、主板限制有测试。
- [ ] 现金不足、持仓不足有测试。
- [ ] 单股 40% 和 8% 止损没有被绕过。

## AI 和记忆

- [ ] LLM 输出经过 schema 或结构化校验。
- [ ] LLM 不能直接下单。
- [ ] LLM 不能直接覆盖账户和规则。
- [ ] 记忆写入经过提案或明确代码路径。

## 外部适配

- [ ] provider 可 mock。
- [ ] 网络失败有错误处理。
- [ ] API key 没有进入日志。
- [ ] 超时、限流、空数据有处理。

## 实盘边界

- [ ] `LIVE_TRADING=false` 仍是默认。
- [ ] 没有新增默认真实 broker。
- [ ] 下单路径经过风控和审计。
- [ ] 危险操作需要人工确认或明确开关。
