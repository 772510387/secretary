## 二、最重要的:那个真 bug 修好了(HAND-02/03)

**问题**:买入后股票被锁成 `todayBuyQuantity`,从无跨日结算 → 纸面持仓**永远卖不出去**。
**修复**:加 `lastBuyTradeDate` + 纯函数 `rollForwardPositionsForTradingDate`;PaperBroker 下单时自动结算,daemon 每个节点/重演开始时也持久化结算。

**真跑验证(离线、无大模型,实测输出)**:
```
day1 买入后: avail:0, todayBuy:100, lastBuy:2026-06-22  现金 19000
当日卖出 -> rejected (insufficient_sellable_quantity)   ← T+1 仍生效
次日卖出 -> filled | 持仓清零 | 现金 20200               ← 隔日可卖,bug 已除
```
