🔄 增长机制（核心！）
每次交易闭环
1. 决策前 → 查询知识库
   "类似场景历史上成功率如何？"
   → 筛选 strategies.json + cases.json

2. 决策时 → 记录依据
   "为什么买？基于哪条策略？"
   → 写入 decision_log.json，关联 strategy_id

3. 决策后 → 记录结果
   "盈亏多少？是否符合预期？"
   → 更新 decision_log.json 的 result 字段

4. 复盘时 → 沉淀经验
   "成功/失败原因？策略需要调整吗？"
   → 成功：写入 cases.json（成功案例）
   → 失败：写入 cases.json（失败案例）+ 更新 strategies.json（调整胜率）

5. 定期 → 知识提炼
   "哪些策略长期有效？哪些失效？"
   → 统计 strategies.json 的胜率/盈亏比
   → 有效策略：提高优先级，写入 documents
   → 失效策略：标记 deprecated，淘汰
